# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute Costs**: $5/user/month
- **Storage Costs**: $2/user/month (assuming 10 GB/user at $0.20/GB)
- **Bandwidth Costs**: $1/user/month (assuming 10 GB of outbound data at $0.10/GB)
- **Total Cost per Active User**: 
  - **Total**: $5 + $2 + $1 = **$8/user/month**

### Pricing Tiers
| Tier          | Price ($/mo) | Features                                             |
|---------------|---------------|------------------------------------------------------|
| Basic         | $10           | Self-hosted setup, basic guidance, email support     |
| Pro           | $20           | All Basic features + advanced security features, priority support |
| Enterprise     | $50           | All Pro features + custom integrations, dedicated support |

### Customer Acquisition Cost (CAC) Range
- **Estimated CAC**: $30 - $50 per user (considering marketing, sales, and support expenses)

### Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**: 
  - Basic: $10
  - Pro: $20
  - Enterprise: $50
- **Average Customer Lifespan**: 24 months
- **LTV Calculation**:
  - Basic: $10 * 24 = **$240**
  - Pro: $20 * 24 = **$480**
  - Enterprise: $50 * 24 = **$1200**

### Break-even Users Count
- **Monthly Fixed Costs**: $2,000 (assumed for infrastructure, support, and overhead)
- **Break-even Calculation**:
  - Break-even Users = Monthly Fixed Costs / (Price - Cost per Active User)
  
Using the Basic Tier:
- Break-even Users = $2,000 / ($10 - $8) = **1,000 users**

Using the Pro Tier:
- Break-even Users = $2,000 / ($20 - $8) = **166.67 users** (rounded to **167 users**)

Using the Enterprise Tier:
- Break-even Users = $2,000 / ($50 - $8) = **50 users**

### Path to $10K MRR
- **Target MRR**: $10,000
- **Users Required**:
  - Basic Tier: $10,000 / $10 = **1,000 users**
  - Pro Tier: $10,000 / $20 = **500 users**
  - Enterprise Tier: $10,000 / $50 = **200 users**

### Summary
- **Total Cost per Active User**: $8/user/month
- **Break-even Users**: 1,000 (Basic), 167 (Pro), 50 (Enterprise)
- **Path to $10K MRR**: 1,000 Basic users, 500 Pro users, or 200 Enterprise users.
