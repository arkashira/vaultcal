```markdown
# Customer Journey Map for VaultCal

## Phase 1: Aware
- **Trigger Event**: User learns about the need for secure self-hosting of contacts and calendar services.
- **Friction Points**: Lack of awareness about self-hosting solutions; confusion about security implications.
- **User Emotions**: Curious but overwhelmed; cautious about security risks.
- **Opportunities to Delight**: Educational content on the benefits of self-hosting; clear comparisons with existing solutions.
- **Metric**: Number of users visiting the VaultCal landing page or engaging with educational content.

---

## Phase 2: Consider
- **Trigger Event**: User actively searches for self-hosted contact and calendar management solutions.
- **Friction Points**: Difficulty in finding reliable information; skepticism about the effectiveness of self-hosted solutions.
- **User Emotions**: Hopeful but skeptical; anxious about making the right choice.
- **Opportunities to Delight**: Detailed guides, user testimonials, and case studies demonstrating successful implementations.
- **Metric**: Number of downloads of product documentation or whitepapers.

---

## Phase 3: Try
- **Trigger Event**: User decides to test VaultCal for self-hosting.
- **Friction Points**: Complexity of setup; lack of clear guidance during installation.
- **User Emotions**: Excited but frustrated; feeling lost during the setup process.
- **Opportunities to Delight**: Step-by-step setup wizard; interactive tutorials or video guides; responsive support during the trial.
- **Metric**: Number of trial installations completed.

---

## Phase 4: Adopt
- **Trigger Event**: User successfully sets up VaultCal and begins using it regularly.
- **Friction Points**: Initial learning curve; potential integration issues with existing tools.
- **User Emotions**: Relieved but still cautious; desire for reassurance about security.
- **Opportunities to Delight**: Regular check-ins from support; proactive tips for optimizing usage; security best practices.
- **Metric**: User engagement metrics (e.g., daily active users, feature usage rates).

---

## Phase 5: Expand
- **Trigger Event**: User sees value in VaultCal and considers additional features or services.
- **Friction Points**: Uncertainty about the value of additional features; potential costs associated with expansion.
- **User Emotions**: Satisfied but cautious; excited about potential enhancements.
- **Opportunities to Delight**: Personalized recommendations based on usage; exclusive offers for existing users; community engagement opportunities.
- **Metric**: Rate of upsells or feature adoption among existing users.
```
